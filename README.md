# 📱 Cleartrip App Automation Framework (Appium + TestNG)

## 🚀 Project Overview
This project is an automated testing framework for the Cleartrip Android application using:

- Appium (Mobile Automation)
- TestNG (Test Execution)
- Java
- Page Object Model (POM)
- Extent Reports (Reporting)

---

## 🧰 Tech Stack

- Java
- Appium
- Selenium WebDriver
- TestNG
- Maven
- Android Emulator

---

## 📋 Prerequisites

Make sure you have installed:

- Java JDK (17 or above)
- Maven
- Node.js (for Appium)
- Appium Server
- Android Studio (Emulator setup)
- Eclipse IDE
